Instrucciones de Compra

  1. Visite University Books y coloque el ISBN: 9798219021350 en la barra de búsqueda.
  2. Siga el proceso de compra y obtenga su código de acceso. El precio es de $50.00 por semestre.

Nota: No compre directamente en CONNECT para obtener el precio especial para este curso.

Acceso Temporal

Si no está listo para la compra, puede activar el acceso temporal de 14 días. Recuerde completar la compra antes de que venza el acceso temporal.
